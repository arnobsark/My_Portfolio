# My Portfolio

A simple HTML & CSS based personal portfolio showcasing projects, skills, and contact information.

## About

This repository contains a responsive portfolio website built with plain HTML and CSS. It is designed to present personal projects, skills, and contact details in a clean and accessible layout.

## Features

- Lightweight static site using only HTML and CSS
- Responsive layout for desktop and mobile
- Sections for About, Projects, Skills, and Contact
- Easy to customize and deploy

## Tech Stack

- HTML5
- CSS3 (Flexbox / Grid)

## Getting Started

To view the site locally:

1. Clone the repository:

   git clone https://github.com/arnobsark/My_Portfolio.git

2. Open `index.html` in your browser.

## Customization

- Edit `index.html` to update your name, bio, and project entries.
- Update styles in `styles.css` (or the CSS file used) to change colors, spacing, and layout.

## Deployment

You can deploy this static site using GitHub Pages:

1. In the repository settings, enable GitHub Pages and set the source to the `main` branch.
2. Your site will be available at: `https://arnobsark.github.io/My_Portfolio/` (depending on your repo settings).
